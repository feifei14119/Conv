# 1. 只要不死都是小事。每一秒都可以从头再来。
# 2. 千万不要出于焦虑，贪图一时的效率把自己逼上困境。面对压力和过劳，人比自己想的要脆弱。
# 3. 不管是亲情、友情还是事业、梦想，任何事任何人都没有资格要求你牺牲身心的健康。
# 4. 不是每个朋友都盼着你好。
# 5. 不要太在意别人的看法。那些有脸让你受委屈的人，你也可以不把他当人。
# 6. 人的生活其实根本不需要那么多虚的东西。比如虚伪的人际交往。
# 7. 自己不保护自己的立场，那么不会有任何人管球你的死活。
# 8. 不要太把自己当个角色。地球几十亿人都在各忙各的，大家各自按照自己喜欢的方式活就行了。（同时也少操心别人的破事）
# 9. 接受自己的不完美。没有人可以完美无缺。
# 10.发现方向错了，立刻回头。
